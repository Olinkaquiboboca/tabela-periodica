Configuração de Imagens

Como configurar as imagens no Cloudinary:

1. Acesse cloudinary.com e faça upload das 118 imagens na pasta elementos/.

2. Use a nomenclatura: elemento_001.jpg, elemento_002.jpg, etc.

3. Em Settings → Security → Allowed fetch domains, adicione https://Olinkaquiboboca.github.io.

4. Use URLs com transformações para qualidade otimizada:
https://res.cloudinary.com/dijqcsy3r/image/upload/w_800,h_800,c_fill,q_90/elementos/elemento_001.jpg

5. Insira estas URLs na tabela elements via SQL Editor: 

INSERT INTO elements (number, symbol, name_pt, category, cloudinary_url)
VALUES
  (1, 'H', 'Hidrogênio', 'nao-metal',
   'https://res.cloudinary.com/dijqcsy3r/image/upload/w_800,h_800,c_fill,q_90/elementos/elemento_001.jpg'),
  (2, 'He', 'Hélio', 'gas-nobre',
   'https://res.cloudinary.com/dijqcsy3r/image/upload/w_800,h_800,c_fill,q_90/elementos/elemento_002.jpg')
  -- ... continua para os 118 elementos
;
